# Time-slip High School Site (ARG templates)

This ZIP contains era-styled HTML pages that progressively get older.
It includes BOTH image-based and text-based "era-aruaru" elements.

## Files / Entry points
- now/index.html (2018+)
- y2014/index.html (2013–2014)
- y2011/index.html (2010–2012)
- y2008/index.html (2007–2009)
- y2004/index.html (2003–2006)
- y2000/index.html + y2000/header.html + y2000/main.html (1999–2001)
- y1996/index.html (1995–1997: mostly text + PDF links)
- y1962/index.html (1960s: PDF-only presentation)
